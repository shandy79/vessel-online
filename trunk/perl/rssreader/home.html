Welcome

In the navigational frame to the left, please select a user and then choose a channel.

The available items from the channel will be displayed in the default order in this frame. You may modify the sort order based on a number of elements from the channel. Please note that not all channels support all the elements available in the list for sorting.

The WebRSS News Reader currently supports the RSS 0.9x and RSS 2.0 formats. Channels in the RSS 1.0 format can be added, but may not display certain elements. Created by Steven Handy.
